# Introduction to Enonic XP

This application contains:
* Tutorial "Introduction to Enonic" (https://developer.enonic.com/learn/intro)
* Sample schemas and data (stored directly in the project root) based on the Headless Movie DB, as well as a simple preview setup. The app also demonstrates use of nested fields (item sets), form localization, dynamically injecting fields across content types (Mixins), optionSets and more.

This application is used in:
* "Introduction to Enonic" tutorial (https://developer.enonic.com/learn/intro)
* "Next.js" tutorial (https://developer.enonic.com/learn/next.xp))
* Solution Template "Headless Demo"
